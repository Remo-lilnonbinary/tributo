The base surface. Warm `paper` by default; `white`/`raised` for elevation, `sunken` for wells, `console` for the sharp operator surface, `ink` for dark panels.

```jsx
<Card variant="white" interactive>Lifts on hover</Card>
<Card variant="console" padding="sm">Operator surface</Card>
```

Variants: `paper | white | raised | sunken | console | ink`. `interactive` adds hover lift. `padding` = `none|sm|md|lg`.
