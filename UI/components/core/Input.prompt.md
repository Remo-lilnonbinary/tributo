Labelled text field, white surface, hairline border, blue focus ring that pairs with the global yellow focus block. Label always visible (never placeholder-only).

```jsx
<Input label="Your question" hint="Plain English is fine." placeholder="Do I need to file a Self Assessment?" />
<Input label="UTR" error="That doesn't look like a 10-digit UTR." shape="sharp" />
```

Props: `label`, `hint`, `error`, `prefix`/`suffix` (icons), `shape` (`pill` citizen / `sharp` console), plus native input attrs.
